# 参加状況ビューア

## この機能でできること

参加校の進捗状況を一覧で確認・管理するための画面です。
メール送信の状況、フォームの回答状況、荷物の入庫状況など、業務全体の進捗をこの画面で把握できます。

## 画面の開き方

1. [ポータル](../portal.html) にアクセスしてパスワードを入力
2. メニューの「参加状況ビューア」をクリック

## タブの説明

画面上部にタブが並んでおり、目的に応じて切り替えて使います。

### 集荷確認タブ

会場（ルート）ごとの参加校一覧を表示します。

- 各学校のメール送信状況、フォーム登録状況、入庫状況がひと目でわかります
- 「この会場にどの学校が参加しているか」「荷物は届いているか」を確認するときに使います

### 仕分表タブ

届いた荷物を会場ごとに仕分けるための一覧です。

- 倉庫での仕分け作業時に印刷して使うことを想定しています

### 参加OPタブ

参加ステータスをマトリクス（表）形式で表示します。

- 学校×ルートの参加状況を一覧で俯瞰できます

### リモート参加タブ

リモート参加の学校を管理する画面です。

### 資料仕分けタブ

資料の仕分け指示書を作成するための画面です。

### 入庫履歴タブ

[入庫記録](guide-receiving.md) で登録された入庫の履歴を検索・閲覧できます。

- 「いつ、どの学校の荷物が何個届いたか」を確認できます
- 学校名やコードで検索できます

### フォーム回答タブ

[荷物登録フォーム](guide-form.md) から送信された回答の一覧です。

- 学校ごとの最新の回答内容が表示されます
- 「この学校は宅配便で何個送ると言っているか」を確認できます

### ダッシュボードタブ

全校の進捗管理と学校情報の編集ができる、最も多機能なタブです。

## ダッシュボードの使い方

### 学校情報の編集

学校名をクリックすると編集画面（モーダル）が開きます。以下の情報を編集できます。

| 項目 | 説明 |
|---|---|
| 案内不要フラグ | ONにすると [集荷案内メール](guide-mail.md) の送信対象から外れます |
| 担当営業 | プルダウンから選択（社員マスタから取得） |
| 担当部署 | 学校側の担当部署名 |
| 担当者名 | 学校側の担当者名 |
| Tel | 担当者の電話番号 |
| メール | 担当者のメールアドレス |
| 倉庫仕分 | 倉庫での仕分け区分 |
| 集荷備考 | 集荷に関する備考・特記事項 |

編集が終わったら「保存」ボタンをクリックしてください。

### フォームURLコピー

ダッシュボードの編集画面から、その学校の [荷物登録フォーム](guide-form.md) のURLをコピーできます。
参加校にメール以外の方法でフォームURLを伝えたいときに使います。

### 荷札生成

編集画面から荷札PDFを生成できます。運営側で荷札を用意したいときに使います。

### 営業SS差異の確認・反映

営業SS（別のスプレッドシート）と学校基本情報に差異がある場合、差異を確認して反映できます。
担当営業やCC希望フラグなどの変更を取り込むときに使います。

## よくある質問

**Q. 画面の表示が古い情報のままになっている場合は？**
→ データはキャッシュ（一時保存）されているため、最新の情報が反映されるまで数分かかることがあります。
すぐに最新の情報を見たい場合は、スプレッドシートのメニュー「システム」→「キャッシュクリア」を実行してからページを再読み込みしてください。

**Q. 案内不要フラグを設定したのにメールが送られた場合は？**
→ フラグを設定した後にメール送信した場合のみ有効です。すでに送信済みのメールには影響しません。

**Q. 営業SS差異とは何ですか？**
→ 営業部門が管理している別のスプレッドシートと、このシステムの学校情報との間に食い違いがある場合に表示されます。担当営業の変更やCC希望フラグの変更などが対象です。
