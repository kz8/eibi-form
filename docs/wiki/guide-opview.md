# 参加状況ビューア

## 概要

参加校の状況を一覧で確認・管理するダッシュボードです。

## タブ一覧

| タブ | 用途 |
|---|---|
| 集荷確認 | 会場別の参加状況・入庫状況を確認 |
| 仕分表 | 仕分け作業用の一覧 |
| 参加○P | 参加ステータスのマトリクス表示 |
| リモート参加 | リモート参加校の管理 |
| 資料仕分け | 資料の仕分け指示書作成 |
| 入庫履歴 | 入庫記録の検索・閲覧 |
| フォーム回答 | 荷物登録フォームの回答一覧（学校別最新） |
| ダッシュボード | 全校の進捗管理・学校情報編集 |

## ダッシュボード編集機能

学校名をクリックすると編集モーダルが開きます。

### 編集可能なフィールド

- 案内不要フラグ
- 担当営業（プルダウン、26社員マスタから取得）
- 担当部署
- 担当者名
- Tel
- メール
- 倉庫仕分
- 集荷備考

### その他の機能

- フォームURLコピー
- 荷札生成
- 営業SS差異の確認・反映
