# 入庫記録

## この機能でできること

届いた荷物の入庫処理を行う画面です。
参加校が [荷物登録フォーム](guide-form.md) で登録した予定個口数と、実際に届いた個口数を記録します。

入庫した情報は [参加状況ビューア](guide-opview.md) の集荷確認タブ・入庫履歴タブで確認できます。

## 画面の開き方

1. [ポータル](../portal.html) にアクセスしてパスワードを入力
2. メニューの「入庫記録」をクリック

---

## 画面構成（4タブ）

| タブ | 用途 |
|------|------|
| **QRスキャン** | 荷札のQRコードをカメラで読み取って自動入庫 |
| **単校** | 1校分の入庫を手入力で記録 |
| **法人一括** | 法人で送られてきた荷物を、配下の各校×各ルートにまとめて入庫 |
| **履歴** | 直近50件の入庫を一覧表示・編集・削除 |

---

## QRスキャンタブ

最も速く入庫できる方法です。荷札に印刷されたQRコード（フォーム画面または資料仕分けタブから生成された荷札）をカメラで読み取ります。

### 使い方

1. 「カメラ起動」ボタンをクリック → カメラアクセスを許可
2. 荷札のQRコードをカメラに向ける（数秒で自動認識）
3. 読み取り内容（学校名・ルート・予定個口）が表示される
4. 入庫個口数を入力 → 「入庫登録」をクリック
5. 続けて次の荷札をスキャン可能

### 仕組み

- 荷札QRには `参加校コード`・`ルート名` が埋め込まれている
- jsQR ライブラリでブラウザ内ローカル処理（外部サーバーには送らない）
- 同じQRを誤って2連続でスキャンしないよう、デバウンス処理あり

### こんなときに使います

- 倉庫で大量の荷物をまとめて入庫処理するとき
- 学校名検索の手間を省いて時間短縮したいとき

---

## 単校タブ

1校分を手入力で入庫するシンプルな画面です。

### 操作手順

1. 検索欄に学校名・コードを入力 → 候補から選択
2. その学校が参加するルート一覧が表示される
3. 該当ルートの行で入庫個口数を入力
4. 「入庫登録」ボタンをクリック

予定個口数も同時に表示されるので、照合しながら入力できます。

---

## 法人一括タブ

法人で送られてきた荷物を、所属校×ルートごとに分配登録できます。

### 操作手順

1. 検索欄に法人名または法人コードを入力 → 候補から選択
2. 配下の全校×全ルートのテーブルが表示される
3. 各セルに入庫個口数を入力（空欄は0扱い）
4. 「一括登録」ボタンをクリック

### こんなときに使います

- 学校法人がまとめて荷物を送ってきて、配下の複数校×複数ルート分を1回で登録したいとき

---

## 履歴タブ

直近の入庫記録を一覧表示します。

### 表示・操作

| 機能 | 説明 |
|------|------|
| 表示件数 | 最大 **50件**（入庫日の新しい順） |
| 状態フィルタ | **入庫済み** / **未入庫** / **すべて** から選択 |
| ソート | 全列でクリックソート可能（学校名・コード・ルート・送付方法・予定/入庫個口・入庫日 等） |
| 編集 | 各行の「編集」ボタンで入庫個口・備考を修正可能 |
| 削除 | 各行の「削除」ボタンで入庫記録を取消（確認ダイアログ付き） |

> 確認ダイアログは `showConfirm` を使った独自実装です（iframe sandbox では `window.confirm` が動作しないため）。

---

## データの保存先

入庫管理シートの「入庫個口」列が更新されます。
QRスキャン・単校・法人一括のいずれも、最終的には同じ入庫管理シートにレコードを書き込みます。

---

## よくある質問

**Q. 入庫個口数を間違えて登録した場合は？**
→ 履歴タブの「編集」ボタンで修正できます。または同じ学校・ルートで再度入庫記録を行えば、上書きされます。

**Q. 予定個口数と届いた個口数が合わない場合は？**
→ 実際に届いた個口数をそのまま入力してください。差異は [参加状況ビューア](guide-opview.md) の集荷確認タブで確認できます。

**Q. 過去の入庫記録を確認したい場合は？**
→ [参加状況ビューア](guide-opview.md) の「入庫履歴」タブで検索できます（最大50件、入庫日新しい順）。
→ 入庫記録画面の「履歴」タブでも同じく直近50件を確認・編集・削除できます。

**Q. QRスキャンが認識しない**
→ ブラウザのカメラアクセス許可を確認してください。読み取り精度はQRコードの印刷品質と照明に左右されます。
→ Tape等で QR が覆われている場合も読めません（荷札にも「⚠ 二次元コードの上にテープを貼らないでください」の注意書きあり）。

**Q. 法人一括で1校だけ登録したい**
→ 単校タブを使ってください。法人一括は複数校をまとめて入力する用途です。

**Q. 削除した入庫記録は復元できる？**
→ 自動復元はできません。誤削除に気づいた場合は、再度入庫登録するか、スプレッドシートの「入庫管理」シートを直接修正してください。
