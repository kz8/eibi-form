# 集荷案内メール送信

## 概要

参加校に対して、荷物の送付方法を案内するメールを送信する機能です。

## 操作手順

1. スプレッドシートを `eibi.soudankai@gmail.com` で開く
2. メニューから集荷案内メール送信ダイアログを開く
3. 送付期限の範囲を指定して対象を抽出
4. 送信先を確認し、送信または下書き作成を実行

## 送信モード

| モード | 動作 | 集荷メール列 |
|---|---|---|
| 通常送信 | 即時送信 | `前期済 MM/DD` |
| 下書き作成 | Gmailの下書きに保存 | `下書き MM/DD` |
| テスト送信 | 自分宛に送信 | 変更なし |

## CC設定

- CC希望フラグが「希望」の学校のみ、社員MailがCCに追加される
- CC希望フラグは営業SSから取得

## 注意事項

- 送信履歴はスクリプト実行アカウントの送信済みに残る
- `eibi.chikoshima@gmail.com` ではなく `eibi.soudankai@gmail.com` でSSを開くこと
