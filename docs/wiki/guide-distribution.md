# 配付物管理

## この機能でできること

進学相談会の各会場で来場者に配る資料や備品を一元管理するための画面です。
顧客（CL）から預かった資料や、自社で用意する配布物について、**どの会場に何部ずつ送るか**を管理できます。

主な機能：

- **品目の登録・管理** — CL名・資料名・セクション（封入/単独配布/置き/備品）を登録
- **会場ごとの部数管理** — 品目ごとに各会場への倉庫部数・事業課部数を入力
- **ルート別の一覧表示** — 物流ルート単位で、どの品目がどの会場にいくつ行くかを横展開で確認
- **xlsx取込** — 顧客からの申込書（Excel）を取り込んで品目と部数を自動登録
- **入庫管理** — 入庫予定日・入庫日・入庫備考を会場ごとに記録

## 画面の開き方

1. [ポータル](../portal.html) にアクセスしてパスワードを入力
2. メニューの「配付物管理」をクリック

---

## 画面構成

画面は3つのタブで構成されています。

| タブ | 用途 |
|---|---|
| **品目一覧** | 登録済みの品目を一覧表示。品目の登録・編集・削除を行うメインの画面 |
| **ルート別** | 物流ルートを選んで、ルートに含まれる会場と品目の部数を横展開で確認 |
| **xlsx取込** | 顧客からの申込書（Excel）をアップロードして品目を一括登録 |

---

## 品目一覧タブ

### 基本操作

1. 画面を開くと、選択中の期に登録されている品目が一覧表示されます
2. **期**を変更すると、その期の品目が自動的に読み込まれます

### フィルタ

| フィルタ | 操作 |
|---|---|
| **期** | 前期・後期を切り替え |
| **セクション** | 封入 / 単独配布 / 置き / 備品 で絞り込み |
| **検索** | CL名・資料名のテキスト検索 |

### 表示される列

| 列 | 内容 | 編集 |
|---|---|---|
| セクション | 封入・単独配布・置き・備品（セレクトボックスで変更可能） | 可 |
| CL名 | 顧客名・提供元名 | 可 |
| 資料名 | 品目の名称 | 可 |
| 外見 | 物理的な外観の説明（A4、白封筒 等） | 可 |
| 倉庫 | 倉庫向け部数の合計 | - |
| 事業課 | 事業課向け部数の合計 | - |
| 合計 | 倉庫＋事業課の総計 | - |
| 会場数 | 部数が入力されている会場の数 | - |
| 入庫予定 | 倉庫への入庫予定日 | 可 |
| 入庫日 | 実際に入庫した日 | 可 |
| 備考 | メモ欄 | 可 |
| 更新日 | 最後に情報を更新した日（自動記録） | - |
| x | 品目の削除ボタン | - |

> 編集可能な列は、セルを直接クリックして入力できます。入力欄から離れると自動的に保存されます。
>
> CL名・資料名を変更すると、その品目に紐づく全ての明細データも連動して更新されます。

### 品目を新しく登録する

1. 「+ 品目登録」ボタンをクリック
2. 期（自動入力）、セクション、CL名、資料名、外見、備考を入力
3. 「登録」をクリック

登録すると、封入セクションの場合は企画設定の封入数に基づいて各会場の部数が自動設定されます。
それ以外のセクションでは、品目だけ登録され、部数は品目詳細パネルで個別に入力します。

### 品目を削除する

品目の行の右端にある「x」ボタンをクリックすると、確認ダイアログが表示されます。
削除すると、その品目に紐づく全ての会場別明細データも一緒に削除されます。

## 品目詳細パネル（会場別の部数入力）

品目一覧の行をクリックすると、画面下部に**品目詳細パネル**が展開されます。
ここで各会場への部数や入庫情報を入力・確認できます。

### 表示される列

| 列 | 内容 | 編集 |
|---|---|---|
| ルート | 物流ルート名 | - |
| 開催都市 | 会場の都市名 | - |
| 開催日 | 会場の開催日（曜日付き） | - |
| 倉庫 | 倉庫に送付する部数 | 可 |
| 事業課 | 事業課に送付する部数 | 可 |
| 入庫予定 | 入庫予定日 | 可 |
| 入庫日 | 実際の入庫日 | 可 |
| 入庫備考 | 入庫に関するメモ | 可 |
| 持ち回り | ルート内で持ち回る場合のグループ番号 | 可 |
| リミット | 倉庫への発送期限 | - |
| 管轄 | 担当支社（東京本社・札幌支社 等） | - |
| 備考 | メモ欄 | 可 |

> 全ての会場（企画設定に登録されている全会場）が表示されます。部数が入力されていない会場にも入力可能で、入力すると自動的に明細データが作成されます。

### 並び順

会場は以下の順番で表示されます：

1. **管轄順** — 東京本社 → 札幌支社 → 仙台支社 → 大阪支社 → 福岡支社
2. **ルート順** — 同じ管轄内では、ルート内の最も早い開催日が先のルートから
3. **開催日順** — 同じルート内では開催日が早い順

ルートごとに背景色が交互に変わるので、ルートの境界が視覚的にわかります。

### 持ち回りについて

同じルート内で、ある会場の資料をそのまま次の会場に持ち回る場合があります。
「持ち回り」列に同じグループ番号（例: 1, 2）を入力すると、同じ番号の会場が持ち回りグループとして管理されます。

- チェーンの先頭の会場にだけ倉庫部数を入力し、後続の会場は0のまま
- テキスト入力なので「A会場から転送」のようなメモとしても使えます

## ルート別タブ

物流ルート単位で品目の配分を確認する画面です。倉庫で発送作業を行う際の確認用に使います。

### 操作

1. 「ルート」セレクトボックスからルートを選択
2. そのルートに含まれる会場が列として横に展開され、品目ごとの倉庫部数が一覧表示されます

### ルートセレクトの並び順

- 管轄ごとにグループ化（東京本社 → 札幌支社 → 仙台支社 → 大阪支社 → 福岡支社）
- 同じ管轄内ではルートの最初の開催日が早い順

### 表示内容

- **ヘッダー行** — 各会場の開催日・都市名・封入数が表示されます
- **データ行** — セクション別にグループ化され、品目ごとの各会場への倉庫部数が表示されます
- **合計列** — 品目の全会場の合計部数
- **備考列** — 品目のメモ

> 倉庫部数が全会場とも0の品目はこの画面には表示されません。

### サマリー

画面上部に品目数・会場数・合計部数が表示されます。

## xlsx取込タブ

顧客からの配付物申込書（Excel）を取り込み、品目と会場別部数を自動登録します。

### 対応するExcelフォーマット

申込書は以下の構造を想定しています：

- **各シート = 1品目**（No.1, No.2, No.3...）
- シート上部に「貴学名」「配付資料名」「納品予定日」「入庫日」「備考」
- シート下部に地区ごとの会場リスト（会場名・開催日・配付希望部数・備考）

### 取込手順

1. **xlsx取込タブ**を選択
2. **期**を選択（品目マスタにこの期が記録されます）
3. xlsxファイルをドラッグ＆ドロップ、またはクリックして選択
4. 「取込実行」をクリック
5. 結果メッセージが表示されます：
   - **追加件数** — 新規に登録された品目の数
   - **スキップ件数** — 既に同名の品目が存在するためスキップされた数
   - **未マッチ都市** — 企画設定に該当する会場が見つからなかった都市名

### 取込の仕組み

- Excelの会場名と開催日を、スプレッドシートの**企画設定**の開催都市・開催日と照合してマッチします
- マッチした会場には、申込書の部数が**倉庫部数**として自動入力されます
- 部数が0の会場は明細データを作成しません（軽量化のため）
- 同じCL名・資料名の品目が既に存在する場合はスキップされます

### 未マッチ都市について

以下のケースでは会場がマッチしません（正常な動作です）：

- 開催日が「未定」「9月開催予定」など具体的な日付が入っていない場合
- 企画設定にまだ日程が登録されていない会場
- 会場名の表記が企画設定と異なる場合

未マッチの会場は、取込後に品目詳細パネルから手動で部数を入力できます。

---

## セクションの種類

品目は4つのセクションに分類されます。

| セクション | 説明 |
|---|---|
| **封入** | ファッションバッグなどに封入して配布するもの。登録時に企画設定の封入数が自動入力されます |
| **単独配布** | 封入せず単独で来場者に手渡しするもの |
| **置き** | 会場の指定場所に積み置きして配布するもの |
| **備品** | 会場運営に関わる備品（ポール、テーブルクロス 等） |

---

## スプレッドシートとの関係

この画面のデータは、スプレッドシートの以下のシートに保存されます。

| シート名 | 内容 |
|---|---|
| **配布物マスタ** | 品目の基本情報（CL名・資料名・セクション・入庫日 等） |
| **配布先明細** | 品目×会場の部数データ（倉庫部数・事業課部数・入庫情報 等） |

> シートを直接編集することもできますが、通常はこの画面から操作してください。

## よくある質問

**Q: 品目を登録したのに品目詳細で会場が表示されない**
A: 品目詳細パネルは企画設定の全会場を表示します。期を正しく選択しているか確認してください。

**Q: xlsx取込で0件追加になる**
A: Excelの「貴学名」「配付資料名」が空のシートはスキップされます。また、同名の品目が既に登録されている場合もスキップされます。

**Q: 部数を入力したら自動保存される？**
A: はい。入力欄から離れると自動的に保存されます。保存中は枠がオレンジ色に、保存完了すると緑色に一瞬変わります。

**Q: 配布先明細シートを作り直したい**
A: シートを削除してから、スプレッドシートのメニュー「配付物管理」→「配付物管理シートを作成」を実行してください。

**Q: 持ち回りの入力方法がわからない**
A: 品目詳細パネルの「持ち回り」列に、同じグループの会場に同じ番号を入力します。例えば新潟→宇都宮→長野と持ち回る場合、3会場とも「1」と入力します。先頭の新潟にだけ倉庫部数を入力し、宇都宮・長野は0のままにします。
